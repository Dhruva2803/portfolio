Crypto-Visionary Price Forecasting System

Tech used: Python, HTML, CSS

Tool:VS code, Google Colab

Functionaluty:

CryptoVisionary is a web-based price forecasting system that allows users to predict the future prices of various cryptocurrencies and commodities like gold and silver. The system is developed using Python (for backend processing and prediction) and HTML/CSS (for frontend display).
Working Process:
1. User Input - The user selects a date for price prediction.
2. Data Processing - The system utilizes historical price data and machine learning models to generate predictions.
3. Prediction Display - The forecasted prices of selected assets (crypto, gold, silver) are displayed on the web interface.

GSM Based Fire Alert System

Tech used: C

Tool:Arduino Studio

Functionality:

The GSM-based fire alert system is designed to detect fire incidents using integrated sensors. When a fire is detected, the system automatically triggers an alert and takes the following actions:
1. Sends an SMS notification – A real-time fire alert message is sent to the owner.
2. Initiates an emergency call – The system places an automated emergency call to the owner or relevant authorities.
3. Shares the location – The system sends the exact location of the fire incident, enabling a quick response.

Tourly Tours and Travels

Tech used: HTML, CSS, JavaScript, Bootstrap

Tool:VS Code

Functionality:

Tourly is a static tours and travels website designed to provide users with essential travel information. It features a user-friendly interface showcasing various travel destinations, tour packages, and itinerary details. The website includes sections for:
1. Destination Listings – Displays popular travel spots with images and descriptions.
2. Tour Packages – Provides details on available tour plans, pricing, and inclusions.
3. Contact & Inquiry Form – Allows users to inquire about travel packages.
